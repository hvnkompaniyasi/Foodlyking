# **App Name**: Foodly King

## Core Features:

- User Authentication: Secure sign-up, login, and profile management for customers and restaurant owners.
- Restaurant Listings & Search: Browse available restaurants with filters based on cuisine type, delivery time, and ratings.
- Dynamic Menu Display: View detailed menus for each restaurant, including dish descriptions, prices, and images.
- Shopping Cart & Checkout: Add and manage items in a shopping cart, with a streamlined process for order placement and payment selection.
- Order Status Tracking: Monitor the real-time status of placed orders from preparation to delivery.
- AI-Powered Food Discovery Tool: Suggest personalized dish recommendations based on user preferences and order history using an AI tool.

## Style Guidelines:

- Primary color: A rich, warm burnt orange (#E0701C) to convey quality, energy, and appetite, subtly hinting at royalty.
- Background color: A very light, warm off-white (#F7F3EF) providing a clean, fresh canvas for content, matching the primary hue.
- Accent color: A deep, bold crimson (#B81425) for strong call-to-action buttons, alerts, and highlighting important elements, providing robust contrast.
- Headline and body font: 'PT Sans', a humanist sans-serif for excellent readability across all text sizes, maintaining a modern and approachable feel.
- Clean, line-based vector icons for navigation and actions, ensuring clarity and a consistent modern aesthetic appropriate for food and delivery themes.
- A responsive, card-based layout for restaurant and menu item displays, prioritizing ease of browsing and content readability on all device sizes.
- Subtle, fluid transitions for menu browsing and cart interactions, enhancing user experience without being distracting.
